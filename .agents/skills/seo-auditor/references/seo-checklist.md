# SEO Technical Checklist

## Core Web Vitals
- [ ] LCP < 2.5s
- [ ] FID < 100ms
- [ ] CLS < 0.1

## On-Page
- [ ] Title tags únicos
- [ ] Meta descriptions en todas las páginas importantes
- [ ] H1 único por página
- [ ] Alt text en imágenes

## Accesibilidad (WCAG 2.1)
- [ ] Contraste de colores suficiente
- [ ] Navegación por teclado
- [ ] Aria labels donde sea necesario
